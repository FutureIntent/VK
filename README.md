# VK
2.p.k

Developer Blogs- tas ir portals ar produkta jaunumiem(news), versijas vesturei(patch notes), iespejai sazinaties ar izstradataju(contacts), atbalstīt projektu(support us) un autorizacijas(log in)/registracijas(register) iespejam. 

Timekļa vietnei ir 3 lomas: admins, autorizets lietotajs un neautorizets lietotajs.

Admins var pievienot publikacijas un redzet privatus vestules no autorizetiem lietotajiem.

Autorizets lietotajs var skatit publikacijas, rakstit komentarus, vērtēt publikacijas un sutit privatus vestules administratoram.

Neautorizets lietotajs var tikai apskatit publikacijas.
